├── water_dataX.csv                  ← Your dataset (place here)

## 🚀 Step-by-Step Setup

### Step 1 — Install Dependencies
Open your terminal / Anaconda prompt and run:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn xgboost lightgbm scipy plotly pymannkendall imbalanced-learn shap joblib streamlit
```

### Step 2 — Place Your Dataset
Make sure `water_dataX.csv` is in the **same folder** as the notebook and app.

### Step 3 — Run the Jupyter Notebook
```bash
jupyter notebook AquaVision_Notebook.ipynb
```
- Run all cells top to bottom (Kernel → Restart & Run All)
- This will train all models and save them to `aquavision_assets/`
- The full analysis takes ~3–5 minutes to run

### Step 4 — Launch the Streamlit App
```bash
streamlit run aquavision_app.py
```
- The app will open in your browser at `http://localhost:8501`
- Use the sidebar to navigate between pages

## 🏆 WQI Scale (CPCB Standard)

| Range | Category | Suitability |
|-------|----------|-------------|
| < 25  | Excellent | Ideal for drinking |
| 25–50 | Good | Suitable for drinking |
| 50–75 | Poor | Treatment required |
| 75–100 | Very Poor | Heavy treatment required |
| > 100 | Unfit for Drinking | Not suitable |
